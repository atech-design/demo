# 🌟 Full Stack Project  

This project is collaboratively built by **Ankit** and **Sujal**.  
- 🎨 **Frontend**: Developed by **Sujal** using **React + Vite**  
- 🛠️ **Backend**: Developed by **Ankit** using **Flask**  

---

## 🚀 Tech Stack  

### 🎨 Frontend (by Sujal)  
- ⚡ React + Vite (with HMR support)  
- 🎭 Modern UI with HTML, CSS, JavaScript  
- 📦 Axios for API communication  
- 🧹 ESLint for code quality  

👉 Currently, two official Vite plugins are available:  
- `@vitejs/plugin-react` → uses **Babel** for Fast Refresh  
- `@vitejs/plugin-react-swc` → uses **SWC** for Fast Refresh  

🔧 For production apps, we recommend **TypeScript** with type-aware lint rules.  
Check out the TS template for details on integrating **typescript-eslint**.  

---

### 🛠️ Backend (by Ankit)  
- 🐍 Flask (Python)  
- 🗄️ MySQL + phpMyAdmin for database  
- 🔐 REST API development  
- ⚙️ Secure authentication & data handling
